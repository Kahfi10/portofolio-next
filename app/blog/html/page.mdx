## HTML (HyperText Markup Language)

adalah kerangka atau fondasi dari sebuah halaman web. HTML digunakan untuk menstrukturkan konten, seperti mendefinisikan mana yang menjadi judul, paragraf, gambar, atau tautan.

## CSS (Cascading Style Sheets) 

adalah desainer yang mempercantik tampilan halaman web. CSS digunakan untuk mengatur visual dari elemen HTML, misalnya mengubah warna teks, jenis huruf (font), latar belakang, dan tata letak (layout).

## JavaScript (JS) 

adalah otak yang membuat halaman web menjadi interaktif dan dinamis. JavaScript memungkinkan Anda untuk menambahkan fungsionalitas seperti animasi, validasi formulir, memuat konten baru tanpa me-refresh halaman, dan merespons tindakan pengguna.

---

**Secara sederhana:**

**-HTML       :** Memberi struktur (kata benda).

**-CSS        :** Memberi gaya (kata sifat).

**-JavaScript :** Memberi perilaku (kata kerja).
